This plugin allows you to display recent events from a specific user or venue and or a single event from Neptix.com.

== Description ==

This plugin pulls events from Neptix.com and will list all your events, or a single event via a simple shortcode. The plugin automatically defines your event date/time, thumbnail and provides a link for your users to buy tickets directly off Neptix.com. The plugin is customizable with a CSS editor built into the plugin to easily style the plugin to fit your websites existing color scheme. 

Visit [BMG MEDIA blog post](http://www.bmgmediaco.com/blog/simplifying-your-websites-coding/) for more information.

== Installation ==

1. Upload the plugin to `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add Neptix shortcode into a page or post
